الدورات والفرص

لا تحتاج إلى تعلم كل شيء في وقت واحد. اختر المهارة المناسبة لمرحلتك وابدأ بالتعلم خطوة بخطوة.

قبل أن تبدأ

ليس الهدف أن تجمع أكبر عدد من الشهادات. الهدف هو أن تتعلم مهارة، وتطبقها، وتبني بها شيئًا حقيقيًا.

  • اختر دورة تناسب مستواك الحالي.
  • لا تنتقل إلى دورة جديدة قبل تطبيق ما تعلمته.
  • حاول بناء مشروع بعد كل مهارة جديدة.
  • احتفظ بمشاريعك في GitHub.

البرمجة وتطوير البرمجيات

تعلم أساسيات البرمجة

ابدأ بتعلم أساسيات البرمجة مثل المتغيرات، الشروط، الحلقات، الدوال، وهياكل البيانات.

المستوى: مبتدئ

التكلفة: مجاني أو مدفوع حسب المنصة.

Git و GitHub

تعلم كيفية حفظ مشاريعك، متابعة التغييرات، والعمل على المشاريع البرمجية.

المستوى: مبتدئ

التكلفة: مجاني.

تطوير الويب

ابدأ بتعلم HTML و CSS و JavaScript لبناء صفحات ومواقع الويب.

المستوى: مبتدئ إلى متوسط.

التكلفة: توجد مصادر مجانية ومدفوعة.

مهارات مهنية مهمة

  • بناء GitHub قوي.
  • إنشاء ملف LinkedIn احترافي.
  • كتابة السيرة الذاتية.
  • بناء المشاريع.
  • التدريب على المقابلات.
  • البحث عن التدريب التعاوني والتدريب الصيفي.

كيف تختار الدورة المناسبة؟

  1. حدد المهارة التي تريد تعلمها.
  2. اعرف مستواك الحالي.
  3. اختر دورة مناسبة لمستواك.
  4. طبق ما تتعلمه في مشروع.
  5. أضف المشروع إلى GitHub.
